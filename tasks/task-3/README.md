# Линтер для async/await

Необходимо написать линтер для проверки в коде отсутствия вызовов `async` функции без `await` внутри блока `if`.

Реализация должна быть в файле `linter.js` - функция `check`. Изучите анализируемый код (`test/code.js`) и тест (`test/index.js`).

Для реализации используйте `acorn` и `zimmerframe`.

Acorn позволяет получить AST, а `zimmerframe` предоставляет метод для удобной итерации по дереву.

Для проверки используйте скрипт `yarn test`.

## Подсказки

В результате работы линтер должен возвращать массив с объектами, описывающими проблемное выражение (поля start и end).

Внимательно изучите AST представленного кода, а затем инструменты.

[Astexplorer](https://astexplorer.net/)

Документация [acorn](https://github.com/acornjs/acorn).

Документация [zimmerframe](https://github.com/sveltejs/zimmerframe).

Примерный алгоритм:
- Прочитать файл с кодом
- Получить AST кода
- Проитерироваться по AST и определить что есть функция которая объявлена с async, но вызвана внутри if без await
